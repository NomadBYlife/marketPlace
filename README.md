## Quick start

Follow these steps to get started with a project:

1. Clone the repository:

   ```
   git clone https://github.com/NomadBYlife/marketPlace.git
   ```
2. Install Dependencies:
   ```  
   cd market-place
   npm install -y
3. To build the production version, run:
   ```  
   npm run build

4. Run server
   ```
   npm start
